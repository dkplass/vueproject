# vueproject

> 電子商務網站實作練習專案，
> 本站郵件訂閱和結帳是模擬並無實際效果，
> 客戶頁面因沒有註冊個人帳號進行識別，購物車可能會已有存在商品，可以清空再測試操作

# Demo

> [Trend](https://dkplass.github.io/vueproject/dist/#/home)

# 介紹

#### 前台
* 首頁形象頁面
* 購物車功能
* 將商品加入最愛
* 商品頁面有總覽和快速分類
* 商品收尋
* 購物結帳流程
* 簡易優惠卷抽獎

#### 後台
* 需管理員帳密驗證
* 商品管理頁面新增、刪除、修改
* 訂單管理頁面，只能進行檢視
* 優惠卷管理頁面新增、刪除、修改
